Summary

넥슨 공식 메이플스토리 API를 활용한 길드원 정보 검색 사이트입니다.

React 라이브러리와 TailwindCSS를 사용하여 아이디어를 실현할 수 있었던 의미있는 1인 프로젝트였습니다.


Technology Stack

주요 기능

◾ 길드원 검색하여 상세정보 확인

◾ 길드원 목록에서 닉네임 클릭으로 상세정보 확인

◾ 장비, 하이퍼스탯, 어빌리티 프리셋 확인

◾ 메일로 의견 보내기

Background

메이플스토리라는 게임에서 중요한 컨텐츠 중 하나는 "길드" 컨텐츠입니다. "길드"는 여러 플레이어가 함께 모여 협력하고 소통하는 단체를 말합니다. 게임 내에서 유리한 위치에 있는, 소위 말해 스펙(캐릭터의 능력치)이 높은 플레이어가 길드 내에 많을수록 길드는 빠른 성장을 하게 되고 높은 레벨의 "노블레스 길드스킬"을 사용할 수 있습니다. 이 스킬은 메이플스토리에서 몬스터를 잡을 때 중요한 부분을 차지합니다. 그렇기 때문에 플레이어들은 메이플GG라는 전적검색 사이트를 통해 길드를 검색하여 길드 구성원들의 스펙을 확인하고 길드 가입 여부를 결정합니다.

길드를 가입하는 경로는 게임 내의 홍보를 보고 가입하는 경우도 있지만, 메이플인벤이라는 사이트에서 길드 홍보글을 보고 가입하는 경우가 많습니다. 그래서 제가 소속된 고래 길드만의 사이트를 만들어 유저가 메이플GG 사이트에 들어가서 길드를 검색해야 하는 번거로움을 줄이고 고래 길드에 대한 정보와 길드원들의 정보를 바로 볼 수 있도록 하여 보다 효율적인 길드 홍보를 위해 제작하였습니다.

Meaning

React로 처음 프로젝트를 진행해 본 경험은 정말 의미 있는 시간이었습니다. 그중 인상 깊었던 경험은 React의 강력한 기능들 중React Hooks를 이용해 효율적으로 상태관리를 하는 것이었습니다. 메일 보내기 기능을 구현할 때 React Hook Form 라이브러리의 useForm 훅을 사용하여 Form 데이터의 유효성 검사를 간편하게 처리할 수 있었던 점과 API와 상호작용하기 위해 썼던 React Query 라이브러리의 useQuery 훅으로 데이터를 가져와 사용할 수 있었던 것이 가장 인상 깊었습니다.

넥슨 메이플스토리 API는 캐릭터의 정보를 가져오기 위해 해당 캐릭터의 고유 id를 API에서 가져온 후, 그 고유 id를 다른 API에 전달하여 쓰도록 되어있는데 이 부분에서 어려움을 겪었으나 라우팅을 관리하기 위한 라이브러리인 React Router에서 제공하는 Link 컴포넌트를 사용해 URL에 고유 id를 전달하고, useParams 훅을 이용해 URL에 전달된 데이터를 추출하여 useQurey 훅의 기능인 쿼리 키에 값을 동적으로 설정하여 API 요청을 보내는 것으로 해결하였습니다. 덕분에 이번 프로젝트는 React의 개념을 다시 한번 잡을 수 있었던 계기가 되었습니다.

다음으로 TailwindCSS와 React를 함께 사용한 경험은 매우 편리하였고 생산성을 높여주었습니다. CSS 파일을 작성하지 않고 클래스를 사용하여 원하는 디자인 요소를 적용할 수 있었던 점과, 반응형 웹을 구현할 때 sm, md, lg, xl 등의 키워드를 통해 간편하게 적용할 수 있었던 점이 저에게는 굉장히 매력적으로 느껴졌습니다.

이러한 경험을 통해 React에 대한 자신감이 높아졌으며, 앞으로의 프로젝트에서도 보다 더 효율적이고 혁신적인 솔루션을 구현할 수 있을 것이라고 기대하고 있습니다. React는 지속적으로 발전하고 있는 기술이기 때문에 끊임없는 학습과 개선이 필요하지만, 그만큼 새로운 도전과 발견이 기다리고 있음을 알게 되었습니다.

ONDOG

URL https://koinoniays2.github.io/ondog/

Summary

OpenWeather API와navigator객체의 getCurrentPosition()메서드를 활용하여 사용자의 현재 위치 기온을 가져와 반려견 산책 온도의 적정 가이드를 제시하는 페이지입니다.



Technology Stack

주요 기능

◾ 현재 위치의 기온 확인

◾ 소형견, 중형견, 대형견으로 분류하여 산책 적정 온도 확인

Background

반려견 천만시대에 강아지를 키우는 보호자들이 늘어나며 반려견의 건강한 사료, 간식, 영양제, 샴푸 등에는 많은 관심이 기울여지고 있지만 많은 보호자들이 반려견을 산책할 때 적절한 온도에 대해 미처 생각하지 못하는 경향이 있습니다. 이러한 이유로 산책 온도의 중요성을 강조하고, 반려견의 건강과 행복에 필수적인 산책 활동에 도움이 되고자 이 페이지를 만들게 되었습니다. 사용자들이 위치를 제공하면 현재 온도를 확인하여 적절한 산책 가이드를 제안합니다.

Meaning

이번 프로젝트는 하루만에 완성한 미니 페이지 프로젝트이지만 바닐라 자바스크립트로 API를 활용했다는 점에서 의미가 있었습니다. 앞서 작성했던 React 프로젝트와 비교를 할 수 있었기 때문입니다. React는 jsx 문법으로 컴포넌트를 정의하고 바로 렌더링 할 수 있었지만 바닐라 자바스크립트에서는 DOM을 직접 조작하여 UI를 렌더링 한다는 차이점을 체감하였습니다. React의 편리함에 의지하는 것이 아니라 바닐라 자바스크립트를 이용하여 API를 활용하고 데이터를 렌더링 하는 과정을 다시 한번 경험하면서 자바스크립트의 기본을 다지는 소중한 기회였습니다.
